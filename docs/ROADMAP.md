# Roadmap

## Phase 1: Public MVP

- Static scenario builder
- Supplier scoring
- Landed-cost and ESG estimates
- Sync manifest export
- Tests and CI

## Phase 2: TSG Integration

- Proxy TSG China Logistics MCP tools through an API
- Import route, tariff, broker, and compliance data
- Add scenario persistence
- Add user workspaces and role-based access

## Phase 3: Sourcing Operations

- RFQ builder and supplier quote intake
- Inspection booking and evidence upload
- Purchase order tracking
- Panama customs and FTZ task boards
- Document vault for certificates, invoices, packing lists, bills of lading, and QA reports

## Phase 4: Climate Device Lifecycle

- Device registry for HVAC assets, IAQ sensors, smart meters, gateways, and retrofit kits
- Firmware, warranty, and installation logs
- Energy and carbon performance tracking
- ESG reports by building, site, portfolio, and supplier

## Phase 5: Network Effects

- Approved supplier marketplace
- Benchmark indexes for landed cost and lead time
- Open schema for climate procurement decisions
- Community-maintained checklist packs for countries, product categories, and certification regimes
